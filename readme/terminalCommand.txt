ls -la - посмотреть содержимое репозитория
cd <name package> - перейти в репозиторий
cd.. - сделать шаг назад по репозиторию
pwd - узнать текущий репозиторий
mkdir <name new package> - создать новую папку в текущем репозитории
open <name package or file> - открить папку или файл в текущем репозитории
touch <name new file> - создать файл в текущем репозитории
mv <name file> <package> - переместить файл в папку
mv <old name> <new name> - переименовать папку или файл
gedit <name package or file> - открыть папку или файл
command + c -  закрыть папку или файл отритые через gedit
